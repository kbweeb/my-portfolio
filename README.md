## Portfolio App

Simple Next.js portfolio. Run locally using npm.

### Prerequisites
- Node.js 18+ (LTS recommended)
- npm 9+ (bundled with Node.js)

### Install
```
npm install
```

### Develop
```
npm run dev
```
Then open:
- http://localhost:3000

Windows PowerShell users: run commands one per line. Example:
```
cd C:\Users\ednaa\portfolio
npm install
npm run dev
```

### Build (optional)
```
npm run build
npm start
```
